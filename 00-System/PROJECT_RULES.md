# 📋 PROJECT RULES — Architecture Rules

## Rule 1: One Object = One File

Слово `haben` существует только один раз. **Все уроки ссылаются на эти файлы.**

## Rule 2: No Duplicates

Перед созданием нового файла анализирую весь репо, обновляю, не создаю.

## Rule 3: Course > Knowledge

Обновляю: 1) Lesson, 2) Vocabulary, 3) Grammar, 4) Exercises, 5) Review

## Rule 4: Everything is Connected

Всё связано. Никаких одноточных структур.

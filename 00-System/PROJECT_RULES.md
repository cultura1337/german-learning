# German Learning OS — Project Rules

## Principles
1. One concept = one file.
2. No duplicate notes.
3. Course comes first, knowledge base second.
4. Every new lesson must include: Lesson, Vocabulary, Grammar, Dialogue, Exercises, Review.
5. Every word must be linked from at least one lesson.
6. Every grammar topic must be linked from at least one lesson.
7. Mistakes are saved as learning material, not deleted.
8. Inbox is the only entry point for new information.

## @GitHub Workflow
When new material arrives:
1. Analyze.
2. Detect existing module.
3. Update instead of duplicate.
4. Add exercises if needed.
5. Link related notes.
6. Keep the repository clean.

## Goal
Build a German Learning OS that can be used daily from an iPhone with minimal friction.

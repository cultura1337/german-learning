# NAMING CONVENTION — How to Name Things

## Folders
```
00-System/              # System docs
01-Course/              # Main course
02-Knowledge/           # Knowledge base (one object = one file)
03-Resources/           # PDFs, audio, video
99-Inbox/               # Temporary (must be empty)
```

## Files
- **lowercase-with-hyphens.md** for general files
- **word.md** for vocabulary
- **rule.md** for grammar
- **README.md** for folders

## Commits
- `system:` System changes
- `content:` New material
- `update:` Updates
- `refactor:` Restructuring
- `delete:` Cleanup

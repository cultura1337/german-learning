# 🇩🇪 German Learning OS

Полноценная операционная система для обучения немецкому языку.

**Не заметочница. Не Obsidian Vault. Архитектурная система.**

---

## 📋 Местораспложение всего

### 📚 Основной курс
**`01-Course/`** — полный структурированный курс
- `A0-Foundation/` — базовые знания
- `A1-Life/` — ситуации из жизни
- `A2-Communication/` — общение
- `B1-Advanced/` — продвинутый уровень

### 🧠 База знаний
**`02-Knowledge/`** — единая база всех знаний
- `Vocabulary/` — все слова (один word = один файл)
- `Grammar/` — все правила (один rule = один файл)
- `Phrases/` — выражения
- `Idioms/` — идиомы
- `Mistakes/` — типичные ошибки

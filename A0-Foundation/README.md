# A0 Foundation — German Learning Foundation

## Module Description

**A0 Foundation** содержит базовые блоки, необходимые для построения всех остальных уроков:

- **Pronouns** — личные местоимения (ich, du, er, sie...)
- **Sein** — глагол "быть"
- **Haben** — глагол "иметь"
- **Questions** — вопросительные слова
- **Basic Sentences** — простые предложения
